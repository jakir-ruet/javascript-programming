Keywords

Keywords are reserved words that are part of the syntax in built-in in the JavaScript

const myVar = 'Hello, Keywords';

Here, const is a keyword that denotes that myVar is a constant.

NB: Keywords can't be used to name identifiers.

The list of keywords available in JavaScript, that shown below;

await break case catch class

typeof var void while with
yield

Identifiers

An identifier is a name that is given to entities such as variables, functions, class, etc.

Naming Rules of JavaScript Identifiers

  1. Identifier names must start with either a letter (a-A), an underscore (_), or the dollar sign ($). JavaScript is the case-sensitive language.

    2. Valid syntax

    const myVar = 'Hello, Identifier';

    const _myVar = 'Hello, Identifier';

    const $myVar = 'Hello, Identifier';
