Operators in JavaScript

SL Operators Short Description
1 Arithmetic Operators mathematical operations between numeric operands, eg a + b
2 Assignment Operators to assign values to variables. eg a = 10
3 Comparison Operators compare two operands and return a boolean value true or false eg a > b
4 Logical Operators combine two or more conditions. eg a || b or a && b
5 Conditional (ternary) Operator it takes three operands: firstly a (?), truthy (:), and finally the expression to execute if the condition is falsy.
6 Bitwise Operator The + operator can also be used to add (concatenate) strings. eg text1 = "John";
